# Network-Security-architecture-
How network security works

Firewall: The Internet connection is secured by a firewall, depicted with a brick wall and flame icon. There are two firewalls shown: one at the main entry point from the Internet and another within the internal network, likely providing an additional layer of security. Firewalls filter incoming and outgoing traffic based on predefined security rules to protect the network from unauthorized access.
Router: Connected to the first firewall, the router directs data traffic between the Internet and the internal network. It ensures that data packets are sent to the correct devices within the office.
Server: The router is linked to a server, which stores and manages data, applications, and services used by the office. The server is a central hub for network resources.
Switches: Two switches are connected to the router and server. Switches act as intermediaries, connecting multiple devices within the network and enabling communication between them. Each switch supports several network computers
Network Computers: Multiple desktop computers are connected to the switches, forming the bulk of the internal network. These computers rely on the switches, server, and router for internet access and resource sharing.
Wireless Access Point: A wireless access point, connected to the second firewall, provides Wi-Fi connectivity. It allows wireless devices, such as laptops, to connect to the network. Two laptops are shown utilizing this wireless connection
